## 🏷️ File Naming Convention

Files are named descriptively based on their content:

- **`lc_`** prefix: LeetCode problems
  - Example: `lc_add_two_numbers_list.cpp` (Add Two Numbers - Linked List)
  - Example: `lc_longest_substring_no_repeat.cpp` (Longest Substring Without Repeating Characters)

- **`cf_`** prefix: Codeforces problems
  - Example: `cf_digit_construction.cpp` (Construct numbers using specific digits)
  - Example: `cf_divisible_groups.cpp` (Count groups divisible by 3)

- **Descriptive names**: Other files clearly indicate their purpose
  - Example: `binary_search_tree.cpp`, `avl_tree_implementation.cpp`

## 🚀 How to Use

### Compiling Individual Files
```bash
# Compile a single file
g++ -o output_name filename.cpp

# Compile with optimizations
g++ -O2 -o output_name filename.cpp

# Compile with debugging symbols
g++ -g -o output_name filename.cpp
```

### Running Solutions
```bash
# Run the compiled executable
./output_name

# Example with input redirection
./output_name < input.txt
```

### Using with Modern C++
Most files use standard C++ features. To compile with C++17 or later:
```bash
g++ -std=c++17 -o output_name filename.cpp
```
